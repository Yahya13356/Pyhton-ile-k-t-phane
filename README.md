---- Python Library Application v1 📚 ----
This little Python app I wrote barely deserves to be called an "application," but for now it creates a temporary list where you can add books, remove them, or display the list.
Since I don’t yet know how to save data, it only holds information temporarily. In the future, I might write the data into a .txt file to make it a permanent library.
To make it feel a bit more feature-rich, I added things like opening a web page or launching a window.
I don’t know who’s ever going to read this, but at least it’s here for now. :)


—— Python 图书馆应用 v1 📚 ——
